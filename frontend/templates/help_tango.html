TANGO (генерируется uRPC)

Требует: сгенерированной библиотеки

Описание

Результатом работы этого генератора является исходный код TANGO Device server на языке C\C++ и файл CMakeLists для сборки адаптера в разных IDE\C компиляторах
Больше информации о том, что такое TANGO Device server, как его написать и как работать с ним, на официальном сайте

Инструкция по сборке

  • Для сборки понадобится CMake, компилятор C\C++ (msvc, redistributable packages на Windows, build-essential на Linux ), пакет TANGO (установить по инструкции: https://tango-controls.readthedocs.io/en/latest/installation/index.html)
  • В директорую с исходными кодами TANGO Device Server поместить заголовочный файл и библиотеку из собранной библиотеки
  • Запустить CMake, а затем сборку по аналогии с запуском CMake и сборкой библиотеки

Инструкция по запуску

  • Для работы программы требуется <имя проекта>.dll и dll-ки от Qt (если Qt прописана в системные пути, то Qt-шные библиотеки должны найтись сами).
    Под linux потребуются библиотеки .so. Путь к библиотеке должен быть прописан в переменной окружения LD_LIBRARY_PATH (см. выше "Сборка под linux"). В некоторых версиях linux после копирования библиотеки нужно вызвать команду

    sudo ldconfig

    для обновления кэша библиотек
  • Можно запускать собранный Tango Device Server. Чтобы посмотреть аргументы командной строки для запуска, запустите програму с флагом

    --help

    инструкцию по работе с TANGO серверами смотрите на сайте TANGO
